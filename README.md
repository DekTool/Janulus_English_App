# janulusingles_app
**Janulus Inglés** es una app interactiva para aprender inglés con 28 matrices progresivas que te ayudan a construir frases de forma natural. Cada palabra muestra su traducción al pasar el cursor, con un diseño limpio y dinámico. Ideal para estudiar gramática visualmente y mejorar tu fluidez paso a paso. ✨
